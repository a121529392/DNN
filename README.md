# DNN
Create a DNN without any keras or tensorflow library

![image](https://github.com/a121529392/DNN/blob/master/image/gui%2Cpng.png)
從下拉選單選擇要測試的資料  
Learning rate :設定學習率  
Convergence condition:設定收斂條件(正確率)  
Network:網路層的架構(隱藏層)  
Training count : 設定訓練次數  
按下OK後開始跑
